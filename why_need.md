## Для чого потрібен цей проєкт

Це **універсальна система автоматизації задач і процесів**.
Вона потрібна, коли є **багато залежних кроків**, які треба виконувати **автоматично, за розкладом або по події**, і при цьому **контролювати збої**.

---

## Що він реально виконує

Проєкт **приймає опис процесу (workflow)** і:

1. **Розуміє логіку**

    * які задачі
    * у якому порядку
    * з якими залежностями

2. **Запускає їх автоматично**

    * по cron
    * по події (HTTP, message, file)
    * вручну

3. **Розподіляє виконання**

    * між потоками
    * між серверами
    * з лімітами і чергами

4. **Контролює виконання**

    * retries
    * таймаути
    * фейли
    * часткове відновлення

5. **Дає спостережуваність**

    * хто що виконується
    * де зависло
    * чому впало
    * історія запусків

---

## Живі приклади використання

### 1. Data / ETL

```text
– забрати дані з API
– очистити
– агрегувати
– записати в БД
– відправити звіт
```

Без scheduler → cron + bash + хаос
З ним → контрольований пайплайн

---

### 2. DevOps / Automation

* деплой
* прогрів кешу
* бекапи
* міграції
* health-check-и

---

### 3. Бізнес-процеси

* обробка замовлень
* виставлення рахунків
* нотифікації
* інтеграції між сервісами

---

### 4. Game / Backend logic

* щоденні квести
* івенти
* перерахунок економіки
* reset-и

---

### 5. Internal platform

Компанії пишуть **такі системи для себе**, бо:

* cron — слабкий
* Jenkins — CI, а не бізнес-логіка
* Airflow — важкий і не завжди підходить

---

## Чим він відрізняється від простого cron

| Cron             | Цей проєкт      |
| ---------------- | --------------- |
| тільки час       | час + події     |
| без залежностей  | DAG             |
| без retry логіки | retry / backoff |
| без UI           | повний контроль |
| без state        | state machine   |

---

## Чому це класний навчальний проєкт

Ти вчишся **не синтаксису Java**, а:

* як будуються **складні backend-системи**
* як керувати станом
* як працювати з конкурентністю
* як проектувати API
* як система поводиться при фейлах
